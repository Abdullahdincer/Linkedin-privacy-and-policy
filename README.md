Privacy Policy for AeroTech Community Management Tool
Last Updated: March 11, 2026

1. Introduction 
This Privacy Policy describes how the AeroTech Community Management Tool ("the Application") handles data accessed through LinkedIn APIs. This application is developed as a technical project for professional community management and educational purposes.

2. Data Collection and Access
The Application requests access to specific data points via LinkedIn OAuth 2.0 authorization, strictly limited to the following scopes:

Organization Analytics: Post engagement statistics, follower growth, and page reach.

Content Management: The ability to draft and schedule posts on behalf of the authorized organization.

Engagement Data: Comments and reactions on organization-owned posts.

3. Use of Data
All data retrieved through LinkedIn APIs is used exclusively for:

Providing analytics dashboards to the organization administrator.

Automating community management workflows.

Educational research on API integration and data visualization.

The Application does not collect, store, or process sensitive personal information of individual LinkedIn members.

4. Data Storage and Third-Party Sharing
No Third-Party Sharing: We do not sell, trade, or otherwise transfer your data to outside parties.

Local Processing: Data is processed primarily in a local environment or a private secure database intended for developer use.

Data Retention: Data is retained only as long as necessary to fulfill the analytics purposes or until the user revokes API access.

5. User Control and Revocation
Users can revoke the Application's access to their LinkedIn data at any time through their LinkedIn account settings under "Permitted Services".
